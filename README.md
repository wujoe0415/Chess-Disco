# 棋逢 Disco

棋逢 Disco 是一款 2D 節奏遊戲，玩家需要跟隨節奏打擊節拍，節拍將會根據西洋棋規則移動，玩家須於旗子移動到正確範圍上時點擊來襲的旗子。在某些時間會出現神秘寶箱，究竟寶箱的技能是好亦或是壞呢？一切都交給節奏舞動吧！
